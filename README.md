# 🛒 Projeto Minhas Compras — .NET MAUI

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como parte das atividades práticas da disciplina de Desenvolvimento Mobile do curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **ETEC Rubens de Faria e Souza – Sorocaba/SP**, no ano de **2026**.

O desenvolvimento foi dividido em **5 agendas**, onde cada uma apresentou a evolução gradual da aplicação.

---

## 🎯 Objetivo

Desenvolver um aplicativo mobile utilizando **.NET MAUI** para gerenciamento de uma lista de compras, aplicando conceitos como:

- Interface em XAML
- Navegação entre páginas
- Persistência de dados com SQLite
- Validação de dados
- Organização em camadas

---

## 🧩 Evolução do Projeto (Agendas)

### 📌 Agenda 1
- Criação do projeto .NET MAUI  
- Estrutura inicial da aplicação  
- Criação da interface básica  

### 📌 Agenda 2
- Criação do modelo de dados (Produto)  
- Implementação do cadastro de produtos  
- Navegação entre telas  

### 📌 Agenda 3
- Implementação do banco de dados SQLite  
- Salvamento de dados localmente  
- Listagem de produtos cadastrados  

### 📌 Agenda 4
- Implementação de edição e exclusão de produtos  
- Melhorias na interface  
- Tratamento de erros  

### 📌 Agenda 5
- Inclusão de **Quantidade** e **Preço**  
- Validação para impedir valores iguais a zero  
- Cálculo do valor total  
- Melhorias finais na aplicação
